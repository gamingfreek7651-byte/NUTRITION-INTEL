Data Engineering

End-to-End Data Pipeline Architecture

From messy scraped raw data listings to clean, optimized, schema-compliant JSON datastores.

01

Ingest Messy Listings

Extract raw data fields from Optimum Nutrition, MuscleBlaze, Nutrabay, Nakpro, and AS-IT-IS CSV files.

02

Standardize & Translate

Normalize Hindi text tags, convert weights (lbs/g to standard kg), and repair price format values.

03

Calculate Analytics Scores

Apply logarithmic scaling for Demand Index, composite Value ratings, and Amazon Best Seller Rank positioning.

04

Generate data.json

Serialize schema-compliant object tree. Offloads heavy computation from frontend for high responsiveness.
